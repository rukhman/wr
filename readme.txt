Backend
1. Открыть терминал и перейти в директорию wolken-reiter-backend
2. Выполнить npm install
3. (Не обязательно) Что бы использовать мои учетные пропустить этот пункт или создать свой файл ".env" по аналогии.
4. Что бы запустить сервер в режиме разработки - выполнить команду npm run dev
Frontend
5. Открыть новый терминал и перейти в директорию wolken-reiter-dev
6. Выполнить npm install
7. Что бы запустить сервер в режиме разработки - выполнить команду npm start
