Database
0. Установить postgresql
1. Создать пустую базу на localhost с именем wrbase или же (не обязательно) что бы использовать мои учетные данные удалить файл ".env" и переименовать .env-sample в .env (в этом случае будет использоваться база данных с бесплатного удаленного сервиса, у нее медленная скорость и ограниченное количество единовременных запросов, поэтому лучше создать свою на localhost).

Backend
2. Открыть терминал и перейти в директорию wolken-reiter-backend
3. Выполнить npm install
4. Что бы запустить сервер в режиме разработки - выполнить команду npm run dev

Frontend
5. Открыть новый терминал и перейти в директорию wolken-reiter-dev
6. Выполнить npm install
7. Что бы запустить сервер в режиме разработки - выполнить команду npm start
8. Сайд доступен на http://localhost:4200/

Доп. инфо:
Сайт еще разрабатывается.
